简单小巧的手势库

移动端开发以来，逐渐的厌倦了寄人篱下的日子！果断自己造了个轮子。
eTouch，是我闲暇之余的一个移动端js简单手势库。

目前满足以下几个功能(请用移动端模式测试)

  • tap 事件的实现
  • 进行单指滑动的实时监控
  • up、right、down、left滑动完毕的回调
  • 并且支持复杂选择器的事件委托

tap的触发以及实时获取手指


var k = 1;
etouch("#box",function(e,touch) {
	console.log(this);
	var log = document.getElementById("boxConsole");
	log.innerHTML = '我被tap了'+ k +'次';
	k++;
});

支持批量绑定

第一个参数为selector(可为复杂选择器)

  • 我是第一个li
  • 我是第二个li
  • 我是第三个li
  • 我是第四个li
  • 我是第五个li
  • 我是第六个li

etouch('.list li',function(e,touch) {
	console.log(this);
	alert(this.innerHTML);
});

也支持事件委托实现

第一个参数直接传#id即可（必须为id）


etouch("#body",".list li span",function(e,touch) {
	alert(this.innerHTML);
});
    我是span

各种方向滑动完成时的回调

我是POX进来滑动试试

	
etouch("#pox",function(e,touch) {
	console.log(e);
	e.preventDefault(); //阻止浏览器默认事件
}).on('swiper',function(e,touch) {
	//既然要监听滑动的回调，首先要在这阻止浏览器默认行为
	//因为上下左右滑动的过程其实就是swiper
	console.log('swiper');
	e.preventDefault();
}).on('up',function(e,touch) {
	$("#coming").append('上滑动
');
	console.log(e);
}).on('left',function(e,touch) {
	$("#coming").append('左滑动
');
	console.log(e);
}).on('right',function(e,touch) {
	$("#coming").append('右滑动
');
	console.log(e);
}).on('down',function(e,touch) {
	$("#coming").append('下滑动
');
});
